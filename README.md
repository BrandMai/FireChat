# FireChat

伴随着移动终端“蓝牙”、“无线局域网”等近距离通讯功能模块的普及，移动终端局域网通讯应用是一个面向无移动网络和WiFi的紧急情况下一个临时通讯需求的应用。移动终端局域网通讯应用的优势在于市面上还未出现一种软件仅通过蓝牙、无线局域网来实现近距离的通话、文字聊天、文件传输以及语音对讲功能，并且方便，快捷，操作简单，所占内存极小。
